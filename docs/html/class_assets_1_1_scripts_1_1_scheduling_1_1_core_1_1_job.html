Detailed Description

An ordered sequence of Operation objects representing a single job.

Tracks execution progress via NextOperationIndex, which advances each time an operation completes inside DESSimulator.HandleOperationComplete. CurrentOperation and IsComplete are derived directly from this index and require no additional state.
