Detailed Description

Direct deserialization target for the Taillard JSON benchmark format.

Field names match the JSON keys exactly to allow zero-configuration deserialization via Unity's JsonUtility or System.Text.Json. After deserialization, JobCount and MachineCount are derived from the matrix dimensions, and GetJobOperations provides the typed operation pairs consumed by DESSimulator.LoadInstance.

int Assets.Scripts.Scheduling.Data.TaillardInstance.machine

Returns the ordered operation sequence for a single job as typed value tuples.

Zips machines_matrix and duration_matrix for the given job index into an array of (machine, duration) pairs in operation order. This is the primary data accessor used by DESSimulator.LoadInstance when constructing Operation objects.

int Assets.Scripts.Scheduling.Data.TaillardInstance.MachineCount
get

Number of machines in the instance, derived from the column count of duration_matrix.

Taillard benchmark instances are always square (jobs × machines = machines × machines), so duration_matrix[0].Length is a reliable column count. Behaviour is undefined if duration_matrix is empty or jagged.
