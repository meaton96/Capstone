◆ BuildFloor()

void Assets.Scripts.Simulation.FactoryLayoutManager.BuildFloor ( DESSimulator simulator)

Builds the factory floor for the given simulator.

Instantiates one MachinePrefab per machine in the simulator, places them on the grid, wires each MachineVisual to its core Machine, and computes the distance matrix.

Call this once per episode from SimulationBridge after the DES has been initialised with a Taillard instance.

Parameters
simulatorThe initialised DES simulator whose Machines array determines how many prefabs to spawn.

Destroys all spawned machine visuals and clears cached data.

Called automatically at the start of BuildFloor so consecutive episodes don't accumulate GameObjects. Safe to call externally for a manual teardown.
