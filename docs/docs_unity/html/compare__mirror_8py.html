Detailed Description

Compares C# DES makespan results against the Python mirror DES implementation.

Both implementations share the same architecture — event-driven simulation with per-machine queues and identical dispatching rules with identical tiebreakers — so any delta in their output indicates a real bug in one of the two implementations, not an expected divergence.

Prerequisites
  1. Run the C# ValidationExporter to produce csharp_makespans.csv and csharp_schedules.json.
  2. Run mirror_des.py to produce mirror_des_makespans.csv and mirror_des_schedules.json.
  3. Run this script from the directory containing all four files:
    python compare_mirror.py
Output
  • A per-rule exact-match summary printed to stdout.
  • An instance × rule comparison table showing OK or a signed integer delta.
  • For any mismatches, the first divergent operation (by job/op key) is identified by diffing csharp_schedules.json against mirror_des_schedules.json.
  • mirror_comparison.csv written to the working directory containing the full merged DataFrame with match and delta columns.
Exit codes
  • 0 — all compared pairs matched exactly.
  • 1 — one or more mismatches found, or required input files were missing.
