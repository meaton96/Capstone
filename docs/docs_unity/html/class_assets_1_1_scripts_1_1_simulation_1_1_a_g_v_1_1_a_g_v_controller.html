Detailed Description

Controls navigation, job assignment, and lifecycle of an AGV.

Implements a Turn-Then-Move model. Routes are planned via the TrafficZoneManager using BFS. Deadlocks are prevented through a reserve-ahead zone system.

Assigns a pickup and delivery task to the AGV.

Resolves the nearest valid docks for the source and target machines, plans a BFS route to the pickup location, and initiates navigation.

Parameters
jobIdID of the job to transport.
pickupPositionTarget world position for pickup.
dropoffPositionTarget world position for dropoff.
sourceThe machine providing the job (null for incoming belt).
dropoffThe machine receiving the job (null for outgoing belt).
Precondition
AGV should be in an Idle state.
Postcondition
State changes to NavigatingToPickup if a route is found; otherwise, returns to Idle.
