Detailed Description

A linear conveyor belt that smoothly moves job visuals between an input end and an output end.

Jobs pack toward the output, advancing automatically when space opens up ahead. Orientation is determined by the transform's forward vector, while the flow direction is toggled via the reverseFlow flag.
