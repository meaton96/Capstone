Detailed Description

ML-Agents Agent subclass that drives job-shop scheduling decisions.

Listens for DecisionRequest events from the SimulationBridge, collects a fixed-width observation vector describing the current machine queue, and maps a discrete action index to a dispatching rule applied via SimulationBridge.Step().

◆ CollectObservations()

override void Assets.Scripts.Simulation.SchedulingAgent.CollectObservations ( VectorSensor sensor)

Builds the observation vector sent to the neural network.

Writes six scalar context values followed by a fixed-width padded block of maxQueueSlots (job-id, duration) pairs. All slots beyond the actual queue length are zero-padded so the vector size remains constant across episodes.

◆ OnActionReceived()

override void Assets.Scripts.Simulation.SchedulingAgent.OnActionReceived ( ActionBuffers actions)

Receives the network's chosen action and applies it to the simulation.

Maps the discrete action index to a dispatching rule via SimulationBridge.Step(), accumulates the returned reward, and ends the episode if the bridge signals completion. The next RequestDecision() call is driven by the HandleDecisionRequired event, not called here directly.

Resets the simulation and begins a new physics episode.

Delegates full episode setup to SimulationBridge.StartEpisode(). The bridge spawns jobs which physically enter machine trigger colliders, causing HandleDecisionRequired to fire and wake this agent when the first scheduling decision is needed.
