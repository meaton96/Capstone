Detailed Description

Heads-up display that polls SimulationBridge each frame and surfaces key episode metrics to the player.

Displays current simulation time, the last dispatching rule applied, total decision count, and a completed-jobs tally. Also owns a time-scale slider that directly modifies Time.timeScale.
