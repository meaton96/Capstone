Detailed Description

Python DES implementation mirroring the C# DESSimulator exactly.

Implements the same event-driven architecture as DESSimulator.cs so that any makespan delta between the two is a definitive indicator of a real bug rather than an expected implementation divergence:

  • Priority queue sorted by (time, sequence_number), matching SimEvent.CompareTo.
  • Per-machine waiting queues with the same idle/busy state machine as Machine.
  • Identical dispatch logic: immediate start if idle, queue append if busy.
  • Tiebreaker on all rules: lowest job_id, matching C# .ThenBy(op => op.JobId).
Usage
python mirror_des.py # runs all ta*.json files in current dir
python mirror_des.py --instance ta01.json # runs a single instance
python mirror_des.py --rules SPT,FCFS # runs a subset of rules
Output
  • mirror_des_makespans.csv — one row per instance/rule, column schema matching csharp_makespans.csv and reference_makespans.csv for direct merging.
  • mirror_des_schedules.json — full operation schedules keyed as "{instance}_{rule_key}", matching the format written by ValidationExporter.RunAndExport.
Exit codes
This script does not currently return a non-zero exit code on mismatch. Use compare_mirror.py to diff its output against the C# results.
