◆ BuildFloor()

void Assets.Scripts.Simulation.FactoryLayout.FactoryLayoutManager.BuildFloor ( DESSimulator simulator)

Generates the factory floor, places machines, and builds navigation boundaries.

Calculates the grid dimensions based on machine count, instantiates the appropriate machine prefabs (single or double-sided), builds aisle walls for NavMesh carving, and computes the spatial distance matrix.

Parameters
simulatorThe active simulation instance containing machine data.
Precondition
simulator must not be null.
Postcondition
The floor is populated with machines, walls, and arrows; the NavMesh is rebuilt.
