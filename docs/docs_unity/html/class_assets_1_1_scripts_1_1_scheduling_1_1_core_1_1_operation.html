Detailed Description

Represents a single operation within a job: process on a given machine for a fixed duration.

Operations are immutable after construction except for StartTime and EndTime, which are set to -1 at creation and populated by DESSimulator during the simulation run. Each operation belongs to exactly one job and targets exactly one machine.
