Detailed Description

Known optimality bounds and bibliographic metadata for a TaillardInstance.

Populated from the JSON metadata block alongside the problem matrices. Used for post-simulation validation: comparing DESSimulator.Makespan against optimum or upper_bound confirms correctness of the dispatching rules. All fields default to 0 if absent from the JSON source.
