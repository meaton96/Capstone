Detailed Description

Continuously orbits the camera around a target Transform.

Each LateUpdate the camera advances its yaw angle by orbitSpeed degrees per real-world second (unaffected by Time.timeScale), then positions itself behind and above the target before looking at it. Assign a target in the Inspector; the camera will log a warning and skip movement if none is set.
