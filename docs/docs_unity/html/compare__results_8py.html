Detailed Description

Cross-validates C# DES output against the job_shop_lib Python reference (v2).

Compares makespan CSVs and full operation schedules produced by the C# ValidationExporter against the job_shop_lib reference generator. Designed to be robust against common data quality issues: duplicate rows from re-runs, missing rules on either side, and large-scale mismatch sets requiring triage.

Prerequisites
  1. Run the C# ValidationExporter to produce csharp_makespans.csv and csharp_schedules.json.
  2. Run the Python reference generator to produce reference_makespans.csv and reference_schedules.json.
  3. Run this script from the directory containing all four files:
    python compare_results.py
Output
  • Coverage report: rows and instances found on each side, and any asymmetric pairs.
  • Overall exact-match / mismatch counts.
  • Per-rule summary showing match rate, average absolute delta, and maximum delta.
  • Instance × rule comparison table showing OK or a signed integer delta.
  • Top-10 worst mismatches by absolute delta, each with the first divergent operation.
  • Diagnosis hints inferring the likely bug class from the sign distribution of deltas and FCFS match rate.
  • Full merged DataFrame saved to comparison_results.csv.
Exit codes
  • 0 — all comparable pairs matched exactly.
  • 1 — one or more mismatches found, or no common pairs could be compared.
