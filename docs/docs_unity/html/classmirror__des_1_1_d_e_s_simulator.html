Event-driven DES with per-machine queues, mirroring C# DESSimulator exactly.

All event handling, dispatch logic, and queue management are intentionally structured to match the C# implementation method-for-method. Any behavioural divergence from the C# DESSimulator is a bug.

◆ current_time

mirror_des.DESSimulator.current_time = 0.0

Current simulation time, advanced on each event dequeue.

Clears the event heap, resets current_time and sequence to zero, restores all machines to IDLE with empty waiting queues, resets all job and operation timestamps to -1.0, and enqueues a JOB_ARRIVED event for every job at t=0. Mirrors C# DESSimulator.Reset.

Active dispatching function, set at the start of each run call.

Sets rule_fn as the active dispatching function, calls reset, then processes events from the min-heap in chronological order until the queue is empty. Mirrors C# DESSimulator.Run.
