Detailed Description

Immutable simulation event with (time, sequence) ordering.

@dataclass (order=True) generates __lt__ from the field declaration order, so time is the primary sort key and sequence is the FIFO tiebreaker — matching the CompareTo logic in the C# SimEvent class. Fields after sequence are marked field(compare=False) to exclude them from ordering.
