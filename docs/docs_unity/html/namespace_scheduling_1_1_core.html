enum Scheduling.Core.DispatchingRule

Composite Primary Dispatching Rules.

For cross-validation against job_shop_lib, tiebreaking is by lowest job_id (matching job_shop_lib's default internal ordering). This ensures identical makespans when the primary sort key is equal.
