Detailed Description

Unity entry point that exports C# makespan results for cross-validation against job_shop_lib.

Attach to any GameObject in the scene. On Play, the component locates all Taillard JSON files in instanceDirectory, runs ValidationExporter.RunAndExport to simulate every instance under every DispatchingRule, and writes csharp_makespans.csv and csharp_schedules.json to outputDirectory. The exported files are then consumed by compare_results.py to diff C# results against the Python job_shop_lib reference.

Setup
  1. Place Taillard JSON files in Assets/Resources/Instances/.
  2. Copy the same files to a regular folder (e.g. Assets/StreamingAssets/Instances/) because ValidationExporter reads directly from disk, not via Resources.Load.
  3. Set instanceDirectory and outputDirectory in the Inspector.
  4. Enter Play Mode.
  5. Run compare_results.py from the output directory:
    cd Assets/StreamingAssets/Validation
    python compare_results.py
