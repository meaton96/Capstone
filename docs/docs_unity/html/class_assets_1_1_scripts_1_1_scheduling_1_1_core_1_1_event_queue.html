Detailed Description

Min-heap priority queue for simulation events, ordered by time with FIFO tiebreaking.

Backed by a SortedSet<SimEvent> which exploits SimEvent's IComparable implementation to maintain heap order without a separate comparator. A monotonically increasing _sequence counter is stamped onto each event at enqueue time to guarantee stable FIFO ordering among events with identical timestamps.
