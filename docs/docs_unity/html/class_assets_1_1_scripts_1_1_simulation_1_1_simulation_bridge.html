Initialises a new episode using the Inspector-configured instance.

Loads the Taillard JSON, resets the DES, optionally builds the visual floor, and advances to the first decision point. After this call, CurrentDecision is populated and IsWaitingForAction is true (assuming the instance produces at least one queuing conflict).

StepResult Assets.Scripts.Simulation.SimulationBridge.Step ( int action)

The core Gymnasium-style step function.

  1. Validates that the DES is actually waiting for a decision.
  2. Maps the action index (0-7) to a composite DispatchingRule.
  3. Snapshots the queue state for logging/visuals.
  4. Calls DESSimulator.ApplyDecision with the chosen rule.
  5. Advances the DES event-by-event to the next decision point (or end).
  6. Computes the step reward from delta-makespan.
  7. If visuals are enabled, plays back the intermediate events.
  8. Returns a StepResult.
Parameters
actionIndex into the composite PDR action space (0-7).
Returns
A StepResult with reward, done flag, and next decision.
