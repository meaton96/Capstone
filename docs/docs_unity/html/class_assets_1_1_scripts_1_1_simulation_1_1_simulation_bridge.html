Detailed Description

Central coordinator between the Unity physics simulation and the scheduling agent.

The bridge owns the episode lifecycle: loading a Taillard instance, spawning physical jobs and machines, reacting to physics events from PhysicalMachine, maintaining a queue of pending scheduling decisions, and surfacing one decision at a time to the SchedulingAgent via OnDecisionRequired. It also computes per-step rewards and fires OnEpisodeFinished when all jobs are complete.
