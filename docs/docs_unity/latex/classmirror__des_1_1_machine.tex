Single shop-\/floor machine with a waiting queue, mirroring C\# \doxylink{classmirror__des_1_1_machine}{Machine}. 

State transitions are driven exclusively by \doxylink{classmirror__des_1_1_d_e_s_simulator}{DESSimulator} via start\+\_\+processing and finish\+\_\+processing, matching the C\# design.
