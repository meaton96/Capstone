\doxysubsection{Detailed Description}
Pure discrete-\/event simulator for \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_job}{Job} Shop Problem (JSP) validation. 

No \doxylink{namespace_assets_1_1_scripts_1_1_scheduling_1_1_unity}{Unity} dependencies — can run standalone or inside \doxylink{namespace_assets_1_1_scripts_1_1_scheduling_1_1_unity}{Unity}. Transit times are zero for benchmark validation. The RL layer will later wrap this and inject non-\/zero transit via AGVs.
