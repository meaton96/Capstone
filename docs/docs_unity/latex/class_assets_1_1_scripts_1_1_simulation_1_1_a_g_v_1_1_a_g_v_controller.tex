\doxysubsection{Detailed Description}
Controls navigation, job assignment, and lifecycle of an \doxylink{namespace_assets_1_1_scripts_1_1_simulation_1_1_a_g_v}{AGV}. 

Implements a Turn-\/\+Then-\/\+Move model. Routes are planned via the \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_factory_layout_1_1_traffic_zone_manager}{Traffic\+Zone\+Manager} using BFS. Deadlocks are prevented through a reserve-\/ahead zone system.
