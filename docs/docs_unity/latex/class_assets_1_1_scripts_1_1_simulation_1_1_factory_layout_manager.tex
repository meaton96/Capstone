Builds the factory floor for the given simulator. 

Instantiates one Machine\+Prefab per machine in the simulator, places them on the grid, wires each \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_machine_visual}{Machine\+Visual} to its core Machine, and computes the distance matrix.

Call this once per episode from \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_simulation_bridge}{Simulation\+Bridge} after the DES has been initialised with a Taillard instance.
