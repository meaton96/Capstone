\doxysubsection{Detailed Description}
Physical anchor for a machine in the Unity scene. 

Manages real-\/time processing via coroutines and delegates visual updates to \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_machines_1_1_machine_visual}{Machine\+Visual}. Supports multi-\/conveyor setups for double-\/sided machines, handling load balancing between belts and proximity-\/based job reception.
