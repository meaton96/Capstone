\doxysubsection{Detailed Description}
Direct deserialization target for the Taillard JSON benchmark format. 

Field names match the JSON keys exactly to allow zero-\/configuration deserialization via \doxylink{namespace_assets_1_1_scripts_1_1_scheduling_1_1_unity}{Unity}\textquotesingle{}s {\ttfamily Json\+Utility} or {\ttfamily System.\+Text.\+Json}. After deserialization, \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_data_1_1_taillard_instance_af2bd50c21201cbe57bd256239db584d6}{Job\+Count} and \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_data_1_1_taillard_instance_a7207f47edf07e7ec6704d88cadebd4c4}{Machine\+Count} are derived from the matrix dimensions, and Get\+Job\+Operations provides the typed operation pairs consumed by DESSimulator.\+Load\+Instance.
