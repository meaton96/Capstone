\doxysubsection{Detailed Description}
Creates, tracks, and updates all jobs in a simulation episode. 

Authoritative record of job states. Transitions are driven by \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_simulation_bridge}{Simulation\+Bridge} events to maintain synchronization with the scheduler.
