\doxysubsection{Detailed Description}
ML-\/\+Agents {\ttfamily Agent} subclass that drives job-\/shop scheduling decisions. 

Listens for {\ttfamily \doxylink{struct_assets_1_1_scripts_1_1_simulation_1_1_decision_request}{Decision\+Request}} events from the {\ttfamily \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_simulation_bridge}{Simulation\+Bridge}}, collects a fixed-\/width observation vector describing the current machine queue, and maps a discrete action index to a dispatching rule applied via {\ttfamily \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_simulation_bridge_a8d81d85edf95a798dd0e6ee4da84a9c1}{Simulation\+Bridge.\+Step()}}.

Builds the observation vector sent to the neural network. 

Writes six scalar context values followed by a fixed-\/width padded block of {\ttfamily max\+Queue\+Slots} (job-\/id, duration) pairs. All slots beyond the actual queue length are zero-\/padded so the vector size remains constant across episodes.
