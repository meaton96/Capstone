\doxysubsection{Detailed Description}
Min-\/heap priority queue for simulation events, ordered by time with FIFO tiebreaking. 

Backed by a {\ttfamily Sorted\+Set$<$\+Sim\+Event$>$} which exploits \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_sim_event}{Sim\+Event}\textquotesingle{}s {\ttfamily IComparable} implementation to maintain heap order without a separate comparator. A monotonically increasing \+\_\+sequence counter is stamped onto each event at enqueue time to guarantee stable FIFO ordering among events with identical timestamps.
