\doxysubsection{Detailed Description}
Represents a single machine on the shop floor. 

Maintains the machine\textquotesingle{}s current \doxylink{namespace_assets_1_1_scripts_1_1_scheduling_1_1_core_a3aebc4d18420ccdbcfb891f2835f01b0}{Machine\+State}, the \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_operation}{Operation} it is actively processing, and a \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_machine_ae56e4992f7199a184ff22c52ae036058}{Waiting\+Queue} of operations that have arrived but cannot yet start. State transitions are driven exclusively by \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_d_e_s_simulator}{DESSimulator} via \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_machine_a5afae06a59815933c3b65c74d889c30c}{Start\+Processing} and \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_machine_af09eca5980967cecce42664fcd3f6b6e}{Finish\+Processing}.
