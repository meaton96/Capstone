\doxysubsection{Detailed Description}
Handles all visual feedback for a single machine on the factory floor. 

Manages mesh colour, overhead \doxylink{namespace_assets_1_1_scripts_1_1_u_i}{UI}, progress bars, and decision flashes. State changes are driven externally by \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_machines_1_1_physical_machine}{Physical\+Machine}.
