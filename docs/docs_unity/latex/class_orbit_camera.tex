\doxysubsection{Detailed Description}
Continuously orbits the camera around a target Transform. 

Each {\ttfamily Late\+Update} the camera advances its yaw angle by {\ttfamily orbit\+Speed} degrees per real-\/world second (unaffected by {\ttfamily Time.\+time\+Scale}), then positions itself behind and above the target before looking at it. Assign a {\ttfamily target} in the Inspector; the camera will log a warning and skip movement if none is set.
