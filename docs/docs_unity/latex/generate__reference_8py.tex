\doxysubsection{Detailed Description}
Generates ground-\/truth makespans and operation schedules from job\+\_\+shop\+\_\+lib for cross-\/validation against the C\# DES simulator. 

Runs every rule in RULES against every instance in INSTANCES using job\+\_\+shop\+\_\+lib\textquotesingle{}s {\ttfamily Dispatching\+Rule\+Solver}, then writes two output files whose structure is kept deliberately in sync with the C\# {\ttfamily Validation\+Exporter} outputs:\+
\begin{DoxyItemize}
\item {\ttfamily reference\+\_\+makespans.\+csv} — one row per instance/\+rule combination, with columns matching {\ttfamily csharp\+\_\+makespans.\+csv} for direct Data\+Frame merging in {\ttfamily \doxylink{compare__results_8py}{compare\+\_\+results.\+py}}.
\item {\ttfamily reference\+\_\+schedules.\+json} — full operation-\/level schedule per instance/\+rule, keyed as {\ttfamily "{}\{instance\}\+\_\+\{rule\+\_\+key\}"{}}, for divergence analysis in {\ttfamily \doxylink{compare__results_8py}{compare\+\_\+results.\+py}} and {\ttfamily \doxylink{compare__mirror_8py}{compare\+\_\+mirror.\+py}}.
\end{DoxyItemize}
