\doxysubsection{Detailed Description}
Python DES implementation mirroring the C\# {\ttfamily DESSimulator} exactly. 

Implements the same event-\/driven architecture as {\ttfamily DESSimulator.\+cs} so that any makespan delta between the two is a definitive indicator of a real bug rather than an expected implementation divergence:\+
\begin{DoxyItemize}
\item Priority queue sorted by {\ttfamily }(time, sequence\+\_\+number), matching Sim\+Event.\+Compare\+To.
\item Per-\/machine waiting queues with the same idle/\+busy state machine as Machine.
\item Identical dispatch logic:\+ immediate start if idle, queue append if busy.
\item Tiebreaker on all rules:\+ lowest {\ttfamily job\+\_\+id}, matching C\# {\ttfamily }.Then\+By(op =\texorpdfstring{$>$}{>} op.\+Job\+Id).
\end{DoxyItemize}
