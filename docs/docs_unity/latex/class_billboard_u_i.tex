\doxysubsection{Detailed Description}
Rotates this Game\+Object each frame to face the main camera. 

Attach to any world-\/space UI element (e.\+g., a floating nameplate or overhead label) that should always be readable from the player\textquotesingle{}s viewpoint. Uses {\ttfamily Late\+Update} so the orientation is applied after all camera movement has settled for the frame.
