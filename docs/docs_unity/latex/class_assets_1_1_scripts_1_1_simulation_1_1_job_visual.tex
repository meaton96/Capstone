\doxysubsection{Detailed Description}
Visual token representing a single job on the factory floor. 

Smoothly interpolates toward targets set by the \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_job_manager}{Job\+Manager}. Movement ownership switches between self-\/driven (lerp), carried (parented to \doxylink{namespace_assets_1_1_scripts_1_1_simulation_1_1_a_g_v}{AGV}), or conveyor-\/driven.
