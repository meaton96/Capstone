\doxysubsection{Detailed Description}
Single operation within a job, mirroring the C\# \doxylink{classmirror__des_1_1_operation}{Operation} class. 

{\ttfamily start\+\_\+time} and {\ttfamily end\+\_\+time} are initialised to {\ttfamily -\/1.\+0} and stamped by Machine.\+start\+\_\+processing during the simulation run, matching the C\# behaviour where {\ttfamily -\/1} sentinel values indicate an unscheduled operation.
