\doxysubsection{Detailed Description}
Cross-\/validates C\# DES output against the job\+\_\+shop\+\_\+lib Python reference (v2). 

Compares makespan CSVs and full operation schedules produced by the C\# {\ttfamily Validation\+Exporter} against the job\+\_\+shop\+\_\+lib reference generator. Designed to be robust against common data quality issues:\+ duplicate rows from re-\/runs, missing rules on either side, and large-\/scale mismatch sets requiring triage.

\begin{DoxyEnumerate}
\item Run the C\# {\ttfamily Validation\+Exporter} to produce {\ttfamily csharp\+\_\+makespans.\+csv} and {\ttfamily csharp\+\_\+schedules.\+json}.
\item Run the Python reference generator to produce {\ttfamily reference\+\_\+makespans.\+csv} and {\ttfamily reference\+\_\+schedules.\+json}.
\item Run this script from the directory containing all four files:\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{python\ compare\_results.py}

\end{DoxyCode}

\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Output}

\begin{DoxyItemize}
\item Coverage report:\+ rows and instances found on each side, and any asymmetric pairs.
\item Overall exact-\/match /\+ mismatch counts.
\item Per-\/rule summary showing match rate, average absolute delta, and maximum delta.
\item Instance × rule comparison table showing {\ttfamily OK} or a signed integer delta.
\item Top-\/10 worst mismatches by absolute delta, each with the first divergent operation.
\item Diagnosis hints inferring the likely bug class from the sign distribution of deltas and FCFS match rate.
\item Full merged Data\+Frame saved to {\ttfamily comparison\+\_\+results.\+csv}.
\end{DoxyItemize}
\end{DoxyParagraph}

\item {\ttfamily 1} — one or more mismatches found, or no common pairs could be compared.
