\doxysubsection{Detailed Description}
Startup menu that lets the user pick a Taillard instance before the simulation begins. 

On {\ttfamily Start}, all {\ttfamily Text\+Asset} files under {\ttfamily Resources/\+\+Instances} are loaded and their names populate a {\ttfamily TMP\+\_\+\+Dropdown}. When the user confirms their selection the chosen asset is pushed to {\ttfamily Simulation\+Bridge.\+Taillard\+Json} and {\ttfamily Start\+Episode()} is called.
