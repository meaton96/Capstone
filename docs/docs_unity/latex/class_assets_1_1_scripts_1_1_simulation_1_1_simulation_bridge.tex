The core Gymnasium-\/style step function. 


\begin{DoxyEnumerate}
\item Validates that the DES is actually waiting for a decision.
\item Maps the action index (0-\/7) to a composite Dispatching\+Rule.
\item Snapshots the queue state for logging/\+visuals.
\item Calls DESSimulator.\+Apply\+Decision with the chosen rule.
\item Advances the DES event-\/by-\/event to the next decision point (or end).
\item Computes the step reward from delta-\/makespan.
\item If visuals are enabled, plays back the intermediate events.
\item Returns a \doxylink{struct_assets_1_1_scripts_1_1_simulation_1_1_step_result}{Step\+Result}.
\end{DoxyEnumerate}
