\doxysubsection{Detailed Description}
Central coordinator between the Unity physics simulation and the scheduling agent. 

The bridge owns the episode lifecycle:\+ loading a Taillard instance, spawning physical jobs and machines, reacting to physics events from {\ttfamily \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_machines_1_1_physical_machine}{Physical\+Machine}}, maintaining a queue of pending scheduling decisions, and surfacing one decision at a time to the {\ttfamily \doxylink{class_assets_1_1_scripts_1_1_simulation_1_1_scheduling_agent}{Scheduling\+Agent}} via {\ttfamily On\+Decision\+Required}. It also computes per-\/step rewards and fires {\ttfamily On\+Episode\+Finished} when all jobs are complete.
