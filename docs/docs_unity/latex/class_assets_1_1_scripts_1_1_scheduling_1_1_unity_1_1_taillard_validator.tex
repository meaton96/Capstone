\doxysubsection{Detailed Description}
\doxylink{namespace_assets_1_1_scripts_1_1_scheduling_1_1_unity}{Unity} entry point for loading and validating Taillard JSP benchmark instances. 

Attach to any Game\+Object in the scene. On Start, or manually via the {\ttfamily "{}\+Run Validation"{}} context menu entry, the component loads a Taillard\+Instance from {\ttfamily Resources/\+\+Instances/\+}, runs constraint checks through Validation\+Runner, executes one or all Dispatching\+Rule variants, and optionally prints a Gantt schedule to the console via DESSimulator.\+Print\+Schedule.

\begin{DoxyNote}{Note}
JSON files must be placed at {\ttfamily Assets/\+\+Resources/\+\+Instances/\+} $<$name$>$.json and referenced by filename without the {\ttfamily }.json extension via \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_unity_1_1_taillard_validator_a2503225f19ffc3f2e69f142c657e366a}{instance\+File\+Name}. 
\end{DoxyNote}

void Assets.\+Scripts.\+Scheduling.\+Unity.\+Taillard\+Validator.\+Load\+And\+Validate (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Loads the configured Taillard instance and runs the full validation pipeline. 

Executes four sequential steps:\+
\begin{DoxyEnumerate}
\item Load the JSON instance from {\ttfamily Resources/\+\+Instances/\+} via Load\+Instance. Aborts early if the file is missing or cannot be parsed.
\item Validate scheduling constraints (precedence, no machine overlap, bound checks) using Validation\+Runner.\+Validate\+Constraints with Dispatching\+Rule.\+SPT\+\_\+\+SMPT. All violations are logged as errors via {\ttfamily Debug.\+Log\+Error}.
\item Run dispatching rules:\+ all rules via Validation\+Runner.\+Run\+All\+Rules if \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_unity_1_1_taillard_validator_a6af62e9c4c5baa1620e068ae69338cd1}{run\+All\+Rules} is {\ttfamily true}, otherwise a single rule via Validation\+Runner.\+Run\+Single. Results are colour-\/coded green /\+ yellow /\+ red based on gap percentage.
\item Optionally print a Gantt schedule to the console if \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_unity_1_1_taillard_validator_aa4fd787eabc293a0c6abe41296aaf692}{print\+Schedule} is {\ttfamily true}, using a fresh DESSimulator run with \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_unity_1_1_taillard_validator_a9c74720d5f136866a9649fb906071a8f}{single\+Rule}.
\end{DoxyEnumerate}

Also callable from the \doxylink{namespace_assets_1_1_scripts_1_1_scheduling_1_1_unity}{Unity} Inspector via the {\ttfamily "{}\+Run Validation"{}} context menu without entering Play Mode.
