\doxysubsection{Detailed Description}
Represents a single operation within a job:\+ process on a given machine for a fixed duration. 

Operations are immutable after construction except for \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_operation_a53d45f1ff0ac549efa8f19bb12c7b07b}{Start\+Time} and \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_operation_a5c826ae3e7644df48c3743d2fd2f5acf}{End\+Time}, which are set to {\ttfamily -\/1} at creation and populated by \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_d_e_s_simulator}{DESSimulator} during the simulation run. Each operation belongs to exactly one job and targets exactly one machine.
