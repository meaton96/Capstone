\doxysubsection{Detailed Description}
Immutable record representing a single scheduled event in the simulation timeline. 

Implements {\ttfamily IComparable$<$\+Sim\+Event$>$} so that instances can be stored directly in a {\ttfamily Sorted\+Set} inside \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_event_queue}{Event\+Queue}. Ordering is by \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_sim_event_a607c1c72df770a64614efd6de3cd4698}{Time} ascending, with \doxylink{class_assets_1_1_scripts_1_1_scheduling_1_1_core_1_1_sim_event_ab56cee8678fd0b8f95a8aa4b18bf4fc8}{Sequence\+Number} as a FIFO tiebreaker for events scheduled at the same instant.
